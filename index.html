Finished Projects

Map of a Mock-up Mall

Map of a Mock-up Mall is a school project made to explore real life uses of Java's GUI features, along with experimentations involving I/O from file to GUI.

To showcase the I/O feature from file to GUI, the administrators were given the option to make new stores, edit description, and set the type of the store, and the shoppers were given the feature to browse stores by type and name, as well as give ratings to each store.

 

Factor+

Factor+ is an IOS application made as a school project in a team of four. The game was made to allow emerging highschool students to be familiarized with the concept of factoring.

After the completion of the game, it was submitted to AppJam, where it won $500 as prize money.
